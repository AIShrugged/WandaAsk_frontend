# Отчёт — 17.03.2026

### 🐛 Фиксы

- **Асинхронный чат (polling)**: Исправлена ошибка
  `react-hooks/set-state-in-effect` — `isSending` переведён на
  lazy-инициализатор `useState(() => ...)` вместо вызова `setState` внутри
  `useEffect`. Монтирующий эффект теперь только выставляет рефы и вызывает
  `schedulePoll()`. Добавлен заголовок `Content-Type: application/json` в
  `sendMessage`.

- **Overflow сообщений в чате**: Длинные строки без пробелов (JSON, URL, токены)
  больше не уходят за границы экрана. Три слоя защиты: —
  `overflow-wrap: anywhere; word-break: break-word` в `.chat-html-content`
  (глобальные стили) — `overflow-x-hidden` на контейнере сообщений — `min-w-0`
  на строке каждого сообщения

- **DemoDropdown / OrganizationDropdown позади контента**: Корневая причина —
  `backdrop-filter: blur()` на `<header>` создаёт `containing block` для
  `position: fixed` потомков. Оба дропдауна перенесены в
  `createPortal(document.body)` с позиционированием через
  `getBoundingClientRect()`.

- **MobileSidebar позади main**: Та же проблема — backdrop + drawer вынесены в
  `createPortal(document.body)`. Кнопка-триггер остаётся в хедере, портал
  рендерит `fixed inset-0 z-40` и `fixed inset-y-0 left-0 z-50` прямо в body.

- **`/auth/organization` не влезает на 390px**: Фиксированные размеры
  `w-[690px] py-[100px] px-[72px]` заменены на адаптивные
  `w-full max-w-[690px] py-8 px-4 md:py-[100px] md:px-[72px]`.

### 📱 Мобильная адаптация

- **Чат (мобильный UX)**: Переработана навигация в стиле WhatsApp/Telegram: —
  `/dashboard/chat` — полноэкранный список чатов — `/dashboard/chat/[id]` —
  полноэкранное окно с кнопкой `← Chats` в хедере — `ChatList` и `ArtifactPanel`
  скрыты на мобильных (`hidden md:flex`) — `ChatList` стал `w-full md:w-[260px]`

- **Хедер дашборда**: Левая часть получила `min-w-0 flex-1` (может сжиматься),
  правая — `flex-shrink-0` (кнопки не сжимаются). Имя и email пользователя
  скрыты на экранах меньше `lg` (`hidden lg:flex`) — виден только аватар.

- **Кнопка `+ add methodology`**: Убраны `ml-6 mr-8` → `mx-4 md:ml-6 md:mr-8`,
  `w-full` → `w-auto md:w-[240px]`. Кнопка больше не выходит за пределы экрана.

- **Кнопка `Create` в Teams**: Аналогичный фикс — `mx-4 md:ml-6 md:mr-8`,
  `w-auto md:w-[170px]`.

- **Страница `/dashboard/follow-ups/analysis/[id]`**: — _Таб-переключатель_
  (Overview / Follow-up / Transcript / Analysis): обёрнут в
  `flex overflow-x-auto`, кнопки получили `flex-shrink-0` — горизонтальный
  скролл вместо переполнения — _Ссылка на митинг_: обёрнута в
  `<span className='truncate'>` с `min-w-0` на контейнере — длинные URL
  обрезаются — _Список участников / гостей_: `flex-row gap-16` →
  `flex-col sm:flex-row gap-8 sm:gap-16` — стакается вертикально на мобиле —
  _Текст деталей события_: `text-[20px]` → `text-base md:text-[20px]`, добавлены
  `min-w-0 truncate`
