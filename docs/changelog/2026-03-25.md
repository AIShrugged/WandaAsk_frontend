### 🔧 Инфраструктура

#### ESLint: принудительный контроль FSD-границ

- **`eslint-plugin-boundaries`** подключён со строгими правилами слоёв:
  `app → widgets → features → entities → shared` (без upward/cross-feature
  импортов)
- **Кастомное правило `local/use-server-in-api`** — ошибка, если в
  `features/*/api/*.ts` отсутствует директива `'use server'`; правило
  автоматически исправляется
- Правило cross-feature импортов выставлено в `warn` (не `error`) — в
  существующей базе 34 нарушения, блокировать push пока нецелесообразно
