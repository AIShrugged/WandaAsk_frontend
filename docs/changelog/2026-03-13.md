# Отчёт — 13.03.2026

### ♻️ Рефакторинг

- **FSD public API**: Все фичи переведены на импорты через `index.ts` вместо
  прямых путей к файлам. `app/page.tsx`, `app/dashboard/page.tsx`,
  `app/dashboard/profile/page.tsx`, `app/dashboard/summary/page.tsx`,
  `features/dashboard/api/dashboard.ts`, `features/calendar/ui/event.tsx`,
  `features/dashboard/ui/RecentChats.tsx` — теперь импортируют через публичный
  индекс фичи.

- **`artifact-panel.tsx`**: Вся логика рендеринга артефактов (switch +
  TYPE_META) вынесена в отдельный компонент `ArtifactCard`. Панель стала
  значительно тоньше (~150 строк удалено).

- **`demo-seed-button.tsx`**: Разбита на три компонента — `DemoDropdown`,
  `DemoOverlay`, `DemoSeedButton`. Добавлен `DemoSeedButtonLoader` (скелетон при
  загрузке) и обновлён `app/dashboard/layout.tsx` для его использования.

- **`summary/page.tsx`**: `KpiCard` вынесен в `features/summary/ui/kpi-card.tsx`
  и экспортируется через `features/summary/index.ts`. Блок графиков-секций
  объединён в компонент `StatsSection`.

### ✨ Новые функции

- **Empty state**: Добавлен компонент `shared/ui/feedback/empty-state.tsx`.
  Пустые состояния теперь отображаются в `FollowUpList` ("No follow-ups yet"),
  `MethodologyList` ("No methodologies yet"), `TeamList` ("No teams yet").

- **Скилл `test`**: Добавлен `.claude/skills/test.md` — скилл для запуска
  unit-тестов (Jest) и E2E-тестов (Playwright) с диагностикой ошибок и
  конвенциями проекта.

### 🧪 Тесты

- **`badge/__tests__/Badge.test.tsx`**: Тесты обновлены под новые CSS-классы
  `bg-accent/15` и `bg-yellow-500/15` для вариантов `success` и `warning`.

- **`follow-up/__tests__/follow-up-list.test.tsx`**: Тест адаптирован под новый
  empty state.

- **`jest.config.mjs` + `jest.setup.js`**: Добавлен мок `next/cache` и полифилл
  `TextEncoder/TextDecoder` — фикс транзитивных импортов через feature-индексы в
  тестах.

### 🎨 Дизайн (a11y)

- **Badge**: Цвета вариантов `primary`, `success`, `warning`, `destructive`
  заменены на контрастные для тёмной темы (WCAG AA). Например: `text-violet-300`
  вместо `text-primary`, `text-emerald-400` вместо `text-green-700`.

- **Input / Textarea / InputDropdown**: Плейсхолдеры повышены с `/50` до `/70`
  для лучшей читаемости. В дропдауне исправлены hover/selected состояния (убран
  зелёный `bg-accent` → `bg-accent/20`/`bg-primary/15`).

- **`globals.css`**: `--muted-foreground` поднят с `52%` до `56%` (контраст 4.41
  → 5.11:1), `--destructive` снижен с `55%` до `49%` (контраст 4.20 → 5.13:1 с
  белым текстом).

- **`input/Error.tsx`**: Цвет ошибки `text-red-700` заменён на
  `text-destructive` для консистентности с дизайн-системой.

### 🤖 Агенты / скиллы

### 🏗️ E2E

- **`e2e/global-setup.ts`**: Добавлена обработка страницы выбора организации
  (`/auth/organization`) — при наличии селектора автоматически выбирается первая
