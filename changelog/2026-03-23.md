### 🐛 Исправления

**Страница не обновлялась после отвязки** (`app/dashboard/calendar`, `app/dashboard/profile`):

- **Причина:** `router.push('/dashboard/calendar')` делает мягкую клиентскую навигацию — Next.js отдаёт закешированный RSC payload, игнорируя `revalidatePath`. Страница profile вообще не была в списке ревалидации.
- **Исправление:** `redirect()` теперь вызывается прямо в server action — Next.js обрабатывает его как hard-навигацию, страница всегда рендерится заново. `router.push/refresh` убраны из клиентского компонента.
