### Ключевые технические паттерны

- **Per-tick `act()` для чейн-таймеров** — каждый `setTimeout` триггерит
  следующий только после flush React-состояния:
  ```ts
  for (let i = 0; i < n; i++) {
    act(() => {
      jest.advanceTimersByTime(52);
    });
  }
  ```
- **`Math.random` mock** — `jest.spyOn(Math, 'random').mockReturnValue(0)` для
  детерминированных задержек в анимациях (PixelRobot)
- **IntersectionObserver с захватом callback** — мок сохраняет `cb` в
  переменную, тест вызывает вручную; ref задаётся через
  `(ref as any).current = el` + `rerender()`
- **`form` атрибут в Button-моке** — нужен для `<button form="id">` за пределами
  формы
- **Fetch-интерцепторы** — `makeRes()` helper вместо `new Response()` (нет в
  jsdom)
